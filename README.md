Homepage
========

This is my personal website.

## Requirements for building

### NodeJS
Get the latest from http://nodejs.org/

### Ruby, Sass & Compass
#### Mac
	sudo gem install sass
	sudo gem install compass

#### Linux
	sudo apt-get install ruby
	sudo apt-get install rubygems
	sudo gem install sass
	sudo gem install compass

#### Windows
Good luck to you!

### Node packages
	npm install

## Building
